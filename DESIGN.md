# Sentinel AI Design System

## Core Product Feeling
- "This is a serious security product."
- NOT an AI product.
- Professional security platform, premium developer tool, mission-critical system.

## Visual Identity
- Minimal, Dense, Elegant, Technical, Operational, Premium.
- NO cyberpunk, neon, glows, hacker aesthetics, robot illustrations, matrix effects, chat UIs.

## Theme Strategy
- Light Mode only for now.

## Color System
- Background Primary: `#FFFFFF`
- Background Secondary: `#FAFAFA`
- Background Tertiary: `#F5F5F5`
- Text Primary: `#111827`
- Text Secondary: `#4B5563`
- Text Muted: `#9CA3AF`
- Accent Blue: `#2563EB` (Hover: `#1D4ED8`)
- Success: `#16A34A`
- Warning: `#D97706`
- Critical: `#DC2626`
- Info: `#0891B2`

## Typography
- Primary Font: Inter (Weights: Regular 400, Medium 500, Semibold 600, Bold 700)
- Monospace: JetBrains Mono
- H1: 40px
- H2: 32px
- H3: 24px
- H4: 18px
- Body: 14px
- Small: 12px
- Code: 13px

## Layout & Spacing
- Max Width: 1440px
- Content Width: 1200px
- Grid: 12 Columns
- Spacing Scale: 4, 8, 12, 16, 24, 32, 48, 64px

## Borders & Shadows
- Border Radius: Small 6px, Medium 8px, Large 12px, Max 16px. Never 24px+
- Shadows: Minimal only `0 1px 2px rgba(0,0,0,0.05)` or `0 2px 8px rgba(0,0,0,0.06)`. NO glowing or glassmorphism.

## UI Components
- Top Navigation Only (no sidebar).
- Tables are first-class citizens (Support search, sort, filters, pagination).
- Empty States: Text only (e.g., "No findings yet. Run a scan to begin."), no illustrations.
- Icons: Lucide icons only.
- Animations: Fade, Slide, Skeleton Loading, Progress Indicators. No particles or parallax.

## AI Visibility Policy
- AI should remain invisible.
- Use "Research Suggested" instead of "AI Agent Thinking...".
- Use "Suggested Finding" instead of "AI Generated Finding".
